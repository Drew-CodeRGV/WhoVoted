Upload District 15 Election Results

CSV Format

Your CSV file should have the following columns:

  • county - County name (e.g., "Hidalgo")
  • precinct - Precinct number (e.g., "101")
  • dem_votes - Democratic votes count
  • rep_votes - Republican votes count

Example:

county,precinct,dem_votes,rep_votes
Hidalgo,101,450,320
Hidalgo,102,380,290
Cameron,201,520,410
