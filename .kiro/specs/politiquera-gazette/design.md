# Design: Politiquera Gazette

## Architecture

The gazette is a static JSON cache + frontend renderer. Stories are pre-generated by Python scripts that query the DB and produce structured JSON. The frontend (`newspaper.js`) renders the JSON as styled articles.

## Cache Format

```json
{
  "generated_at": "2026-05-10T23:00:00",
  "election": "McAllen ISD Bond Election",
  "election_date": "2026-05-10",
  "stories": [
    {
      "id": "turnout_summary",
      "headline": "12,847 McAllen Voters Cast Ballots in Bond Election",
      "lede": "Turnout exceeded expectations for a special election...",
      "body": "...",
      "data": { "total_voters": 12847, "registered_voters": 89234, "turnout_pct": 14.4 },
      "type": "turnout"
    }
  ]
}
```

## Cache Builders

Each election has a dedicated cache builder script:
- `deploy/cache_misdbond2026_gazette.py`
- `deploy/cache_elsa2026_gazette.py`
- `deploy/generate_statewide_gazette_cache.py` (for the main map)

Pattern:
```python
def build_gazette(election_date, election_name, boundary_filter):
    stories = []
    stories.append(build_turnout_story(conn, election_date, boundary_filter))
    stories.append(build_party_story(conn, election_date, boundary_filter))
    stories.append(build_demographic_story(conn, election_date, boundary_filter))
    stories.append(build_precinct_story(conn, election_date, boundary_filter))
    stories.append(build_new_voter_story(conn, election_date, boundary_filter))
    return {'generated_at': now, 'stories': stories}
```

## Frontend: `public/newspaper.js`

- Opens as a slide-in panel (right side)
- Renders stories as styled article cards
- Paywall: if `window.__subscribed === false`, blur all content and show overlay
- Mobile: single-column layout

## Paywall Integration

```javascript
// In newspaper.js
if (!window.__subscribed) {
    gazettePanel.classList.add('paywalled');

}
```

## Story Generation Logic

Stories are generated by querying the DB and substituting values into narrative templates. No LLM involved — pure data substitution.

Example:
```python
def build_turnout_story(conn, election_date, zip_filter):
    total = conn.execute("SELECT COUNT(DISTINCT vuid) FROM voter_elections WHERE election_date = ?", (election_date,)).fetchone()[0]
    registered = conn.execute("SELECT COUNT(*) FROM voters WHERE zip IN (...)", zip_filter).fetchone()[0]
    pct = round(total / registered * 100, 1)
    return {
        'id': 'turnout_summary',
        'headline': f'{total:,} Voters Cast Ballots',
        'lede': f'Turnout reached {pct}% of registered voters.',
        'body': f'A total of {total:,} voters participated...',
        'data': {'total': total, 'registered': registered, 'pct': pct}
    }
```
